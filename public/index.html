🧪 Water Chemistry API

Precision brewing water chemistry calculation API with 100% Bru'n Water compatibility.

📡 API Endpoints

GET /api/salts

Returns all available brewing salts with their chemical properties.

curl https://water-chemistry-api.vercel.app/api/salts

GET /api/profiles

Returns standard brewing water profiles (Pilsen, Burton, Dublin, etc.).

curl https://water-chemistry-api.vercel.app/api/profiles

POST /api/calculate

Performs comprehensive water chemistry calculations including mash pH, salt additions, and acid requirements.

Send JSON payload with sourceWater, grainBill, and volumes.
